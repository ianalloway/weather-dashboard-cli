# Contributing

Thanks for helping improve this project.

## How to contribute

- Open an issue to discuss larger changes before you invest a lot of time.
- Keep pull requests focused and describe **what** changed and **why**.
- Follow the development setup in the README and run the project’s tests or linters before opening a PR.

## Security

Please do **not** report security vulnerabilities in public issues. See [`SECURITY.md`](SECURITY.md) for how to reach the maintainer privately.
